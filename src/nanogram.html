Welcome to Nanogram.

The original idea of social media was to share pictures and see what your friends were doing. These days, seeing your friends' pictures and engaging with friends has almost disappeared. The only engagement that social media strongly promotes is sharing reels. A feed that used to be mostly your friends' posts is now filled with random posts from the internet that an algorithm decided are worth your time.

This is a vibe-coded attempt to host my profile outside social media. This project is not better than social media, and it is not meant to replace it. It is simply an expression of my thoughts on the current state of social media.

In the future, I want to add a feature that links my friends' profiles, so anyone can visit them like any other social platform. With nanogram, you have full freedom over customization. You can choose any background or animation for your profile.

The site collects login attempts, so if you want to communicate with me privately, you can submit messages as password attempts, and I might read them now and then.

The site is slow because all media is stored at the highest possible quality and encrypted with a password. The site decrypts only after the one and only password is entered. If you do not have the password, talk to me in real life :-)

This project is now open source on GitHub .

Configure your build from a config file and refer to the GitHub repository for full setup, recovery, and build instructions.
